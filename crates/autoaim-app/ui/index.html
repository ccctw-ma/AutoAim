How to use

Four-step offline review

  1. Choose a frame JSONL file, or use the bundled sample file.
  2. Run validation to catch missing grouping fields or invalid boxes.
  3. Run evaluation to compute target suggestions and summary metrics.
  4. Write event JSONL when you need review-only inference results.
